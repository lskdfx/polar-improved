# polar-improved

A grapher for polar functions built using Plotly/Dash for frontend and sympy, numpy, and scipy for math manipulation.

## Setup
1. Install Python
2. `python3 -m venv .venv`
3. Activate the venv (it differs by shell): `source .venv/bin/activate`
4. `pip install -r requirements.txt`
5. `python3 src/app.py` (web view) or `python3 src/native.py` (native debug view using matplotlib)
